##

:::note
CoPRO is essentially a paired-end variation on PRO-cap and PRO-seq in which each end of the paired-end library fragment encodes the TSS (PRO-cap) or the RNA Pol II active site (PRO-seq). Analysis of the single end variants will more or less follow the CoPRO tutorials except the user should switch to the 5' end of Read 1 encodings for all single-end variants.
:::
