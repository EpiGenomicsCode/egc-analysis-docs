## Download ScriptManager (v0.15):

The current version of ScriptManager is available for download **[here][jar-download]**. Make sure you have [Java installed][java-install].

The file `ScriptManager-v0.15.jar` should be placed someplace locally accessible. For example on Mac OS on the Desktop (Permissions will need to be accepted) or someplace in your home directory (`Macintosh_HD:Users/userID/ScriptManager`)

[jar-download]:https://github.com/CEGRcode/scriptmanager/releases/download/v0.15/ScriptManager-v0.15.jar
[java-install]:https://pughlab.mbg.cornell.edu/scriptmanager-docs/docs/#-dependencies-just-java
